# Arduino-Uno-Enclosure

My project is an Arduino enclosure! It has gaps for places where you insert cables, and also has a removable lid. It is fully parametric, meaning that if you wanted to put in a different arduino model with different dimensions, it is very easy to do so as you just have to change some numbers. I decided to make this because I am always worried about potentially damaging my arduino whenever I use it, so I thought why don't I create my own enclosure to protect it? This project has taught me more about modeling in fusion, as I learned how to use parameters.
